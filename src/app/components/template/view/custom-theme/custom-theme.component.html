Colors

Change the palette color of template.

Access to: src/app/assets/css/palette.sccs

Override the Bootstrap variables colors

  • $primary: #007bff;

  • $secondary: #6c757d;

  • $success: #28a745;

  • $info: #17a2b8;

  • $warning: #ffc107;

  • $danger: #dc3545;


And then use this page to http://mcg.mbitson.com/ to generate new theme for angular material.

Override the Angular Material variables colors

  • md-matprimary: #007bff;

  • md-matsecondary: #6c757d;

  • md-matsuccess: #28a745;

  • md-matinfo: #17a2b8;

  • md-matwarning: #ffc107;

  • md-matdanger: #dc3545;
